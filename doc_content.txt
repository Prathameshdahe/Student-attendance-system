Smart Attendance System
Comprehensive Technical Documentation
1. Project Overview
The Smart Attendance System is an Android-based application designed to automate and secure student attendance using QR codes, GPS geofencing, and biometric face verification. The system prevents proxy attendance and provides real-time monitoring for administrators.
2. Problem Statement
Traditional attendance systems are vulnerable to proxy marking, manual errors, and lack real-time monitoring. Institutions require a secure automated system that validates presence through multiple verification layers.
3. Proposed Solution
The system integrates three verification layers:
1. Dynamic QR Codes (daily refresh)
2. GPS Geofencing
3. Face Verification
These combined mechanisms ensure only legitimate students are marked present and remain within campus boundaries.

7. Why Not Google ML Kit
Google ML Kit works well for basic detection but struggles with:
• slower processing on low-end devices
• limited face recognition capability
• accuracy issues in crowded environments
Better Alternatives:
• InsightFace (best accuracy)
• FaceNet
• Mediapipe
8. Workflow Process
Phase 0: Midnight QR generation for each student
Phase 1: Student check-in (QR + Face verification)
Phase 2: Geofence monitoring during the day
Phase 3: Checkout verification
Phase 4: Attendance calculation and report generation
